### Cách dùng

Mỗi đề có file `CODING_GUIDE.md` — đưa thẳng vào AI kèm theo:
- `CODING_GUIDE.md`
- File `.sql` trong thư mục `Given`
- File `debai.txt`
- Ảnh ERD

> ⚠️ Code trong guide còn sai một số chỗ — dùng làm context cho AI hỏi thêm, không copy blind
