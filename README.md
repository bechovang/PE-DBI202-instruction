### Cách dùng (QUAN TRỌNG)

Khi làm mỗi đề, **không dùng mỗi CODING_GUIDE** — phải dùng đủ bộ:

* `CODING_GUIDE.md`
* File `.sql` trong thư mục `Given`
* File `debai.txt`
* Ảnh ERD

👉 Nên đưa **tất cả vào AI** để hỏi + check xem mình làm đúng hay sai.

---

### Lưu ý quan trọng khi dùng

* ⚠️ **CODING_GUIDE sai khá nhiều**

  * Không được copy code trực tiếp
  * Chỉ dùng làm **context tham khảo**

* 📌 **Câu 1 (Q1)**

  * Làm dựa trên **ERD (sơ đồ)**, KHÔNG phải database có sẵn
  * Không chạy SQL được
  * Thực chất là: **chuyển ERD → SQL**

* 📌 **Database**

  * Lấy từ folder `Given`

* 📌 Nếu thấy:

  * ERD ≠ Coding Guide → **tin ERD**
  * Coding Guide chỉ để tham khảo

---

### Quy trình làm chuẩn

1. Mở:

   * ERD
   * debai.txt
   * file SQL (Given)

2. Làm từng câu:

   * Không hiểu → hỏi AI
   * Làm xong → **kêu AI check lại**

3. Không phụ thuộc guide

---

### Cấu trúc mỗi đề

```
Paper X/
├── CODING_GUIDE.md   ← hướng dẫn từng câu (có thể sai)
├── debai.txt / .pdf  ← đề bài
└── Given/            ← script tạo database
```

---

### Tip nhỏ

* Câu 1 thường **dễ nhưng dễ sai**
* Nếu thấy "nghi nghi" → check lại logic ERD
